## 🚀 Jak Zacząć?

```bash
# 1. Sklonuj repozytorium
git clone https://github.com/twoja-nazwa/bookself.git

# 2. Zainstaluj zależności
npm install

# 3. Skonfiguruj .env
cp .env.example .env

# 4. Uruchom!
npm run dev
